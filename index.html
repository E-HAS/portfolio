🙋‍♂️ 소개

저는 안정성과 확장성을 고려한 시스템 아키텍처 설계를 중시하며, 실사용자의 관점에서 문제를 해결해 나가는 백엔드 중심의 풀스택 개발자입니다.

주로 Java 기반의 Spring Boot와 .NET 환경에서 API 설계, DB 모델링, 서버 운영 및 성능 최적화까지 전 과정을 책임감 있게 수행해왔으며, 다양한 외부 시스템 연동과 로그 분석, 실시간 처리 경험을 통해 복잡한 비즈니스 로직을 효과적으로 구현해 본 경험이 있습니다.

“사용자에게 안정적인 서비스를 제공한다”는 원칙 아래, 기능의 완성도뿐 아니라 유지보수성과 협업의 효율성까지 고민하며 일합니다.
